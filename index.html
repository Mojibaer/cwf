What is CopyWithFriends?

Install the CopyWithFriend tool and start generating your first code snippets easily.

Insights

CopyWithFriend is a robust tool developed with Go, designed to streamline code sharing and collaboration. It allows users to create text files and save them on a server, making them accessible to other users. This facilitates seamless collaboration, as all participants can view and modify the shared files, enhancing productivity and speeding up project timelines.

Coming next

Looking forward, we are excited to expand CopyWithFriend with new features that will provide enhanced file management options, improved user interfaces, and increased security measures to ensure your data remains safe. Commands List

Send content to server and save it in a single file.

echo "Hello Clipboard!" | cwf testfile Copied! Output:
Saved to: testfile

Get content of a saved file from server.

cwf testfile Copied! Output:
Hello Clipboard!

Send content to server and save file in a subdirectory.

echo "Hello Clipboard from subdirectory!" | cwf testdir/testfile Copied! Output:
Saved to: testdir/testfile

Get content of a saved file in a subdirectory from server.

cwf testdir/testerfile Copied! Output:
Hello Clipboard from subdirectory!

Delete single file

cwf -d testerfile Copied! Output:
Deleted file: testerfile

Delete single file in subdirectory

cwf -d testdir/testfile Copied! Output:
Deleted file: testfile

Delete all files in subdirectory

cwf -d testdir/ Copied! Output:
Deleted directory: testdir

List files and directories in configured base directory.

cwf -l Copied! Output:
Type Name Modified
Dir testdir 2006-01-02 15:04:05
File testfile.cwf 2006-01-02 15:04:05

List files and directories in a subdirectory.

cwf -l testdir Copied! Output:
Type Name Modified
File testfile.cwf 2006-01-02 15:04:05
